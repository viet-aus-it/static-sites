# VAIT Static Sites monorepo

This repo contains VAIT Static sites apart from the [homepage project](https://github.com/viet-aus-it/homepage).
Each folder in this directory will represent a site that links to the same subdomain. (e.g., `chat` will link to `chat.vait.au`).

## Directory

| Dir  | Site                                         |
| -----| -------------------------------------------- |
| chat | [https://chat.vait.au](https://chat.vait.au) |
| fb   | [https://fb.vait.au](https://fb.vait.au)     |
| linkedin | [https://linkedin.vait.au](https://linkedin.vait.au) |
| yt   | [https://yt.vait.au](https://yt.vait.au)     |
